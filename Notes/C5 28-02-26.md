```javascript 

// Create Http Server
import exp from "express";
// the express module internally contains http server
const app = exp();

// set a port number

const port = 6767;
// assign port number to HTTP server
app.listen(port, () => console.log(`server listning at port : ${port} ...`));

// Create API (REST API -> REpresentational State Transfer)
// route to handle GET req from client
// user api's
app.get("/users", (request, response) => {
    // semd response to client
    response.json({ message: "The user data is sent!" });
});
// route to handle POST req from client
app.post("/users", (request, response) => {
    // semd response to client
    response.json({ message: "The data is sent to user!" });
});
// route to handle PUT req from client
app.put("/users", (request, response) => {
    // semd response to client
    response.json({ message: "The users are modidied!" });
});
// route to handle DELETE req from client
app.delete("/users", (request, response) => {
    // semd response to client
    response.json({ message: "The user is deleted!" });
});


```

### Steps to create backend
1. Genrate package.json File ``` npm init -y ``` to initialize it
2. Create HTTP Server
    - Install Express module ``` npm install express ```
    - Import Express Module
3. Define routes 
    - give them request and response definition -> get, post, put, delete!


```javascript
// Create Http Server
import exp from "express";
// the express module internally contains http server
const app = exp();

// set a port number
// Test data (will be replaced with real db)

const port = 6767;

    // semd response to client
    response.json({ message: "The users are modidied!" });
```
